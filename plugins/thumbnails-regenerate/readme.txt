Regenerate Thumbnails - New Version, fixed all bugs. Delete and regenerate all thumbnails for your image attachments.

== Installation ==

1. Download plugin zip file to your computer and extract in a folder
1. Upload `thumbnails-regenerate` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
